# Polarzero Portfolio

This site is my personal portfolio, showcasing work, experiments, writing and education.

## Stack

- React 19 with Vite 7 for the app shell
- Tailwind CSS and shadcn/ui for styling
- Biome for linting and formatting

## Run Locally

```bash
pnpm i
pnpm dev
```
